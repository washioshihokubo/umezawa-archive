見てみたいモノ。梅澤美波

2017.03.27 23:09

COPIED

 
 
 
 
 
 
 
みなさまこんにちは！！
 
 
 
 
乃木坂46 ３期生の
梅澤美波です！！
 
 
盛りだくさんです
 
 
 
 
 
レアデコ出し
レアとかいって載せちゃったらレアじゃなくなっちゃう？？( .. )
 
髪暗くしたのに伝わらない( .. )
 
 
 
 
 
 
 
 
個人PV、見ていただけましたか？
 
 
まだまだ未熟な私たち３期生にも
個人で撮って頂けるなんて思ってもなくて
聞いた時は本当に驚きました。
 
不安でいっぱいだった( .. )
 
 
 
 
初めての個人PV、
めちゃくちゃ緊張したんです！！
 
でも、監督さんもスタッフさんも
とても優しくて楽しかったなあ( ¨̮ )
 
 
私は、３期生の中でも、先輩方の中でも
個人PVを撮るのが１発目で、
ほかの３期生のメンバーよりも
だいぶ早く撮ることになったので
余計緊張しっぱなしでした、、。
 
 
 
個人PVで着てる服と、
会議室のシーンの後ろにかかってる服は
ぜんぶ私服です〜！！
 
 
 
監督さんは、
深川麻衣さんの強がる蕾のMVを
撮られた真壁幸紀監督さんと、
人形映像作家さんの飯塚貴士さんです☺︎
 
とても優しい方でリラックスして撮影に望めました（ ｉ _ ｉ ）
 
 
 
 
緊張したけどものすごく楽しくて
移動の合間にスタッフさんとマネージャーさんとご飯食べに行ったりとか
撮影全部終わってスタッフさん方みんなでご飯食べてお話ししたりとかして
 
素敵な方たちに囲まれてる〜
幸せだ〜
って感じながらの撮影でした( ¨̮ )
 
 
コマ撮りとかもあったりして、
本当に作るの大変だと思うんです、
でも、こんなに素敵な個人PVができて本当に幸せです（ ｉ _ ｉ ）
 
 
個人的お気に入りシーンは
ど真ん中に真剣な顔して立って画面を見つめてるとこ( ¨̮ )（笑）
頑張ったシーン( ¨̮ )
 
 
オフショット( ¨̮ )
 
 
 
１枚目、２枚目は
噴水のある公園で撮影したんだけど
めちゃくちゃ天気よくて気持ちよかったの◎
 
３枚目は、原宿の竹下通り入口！
人が多すぎて恥ずかしかったけど楽しかったな〜
 
最後はお人形のマネージャーさん、轟木さん！
１日ありがとうね◎
 
 
 
 
 
 
そしてsamuraiELOさん
見ていただけましたか？？
 
 
オフショット！！
 
 
 
ももっぴとのカット！！
このお洋服、実は背中がめちゃくちゃガッツリあいてるんです！
可愛かったなあ
 
 
他にも服はメンズ服を着たりと、とてもかわいかった( ¨̮ )
本当に幸せだった〜お洋服だいすき！！
 
 
 
 
ファッションのお仕事、
本当に幸せでした。お洋服だいすき！
 
これから少しずつ増やしていけたらいいな〜
って、思いました。
 
がんばる！
 
 
 
 
 
 
 
 
 
コメントでかけてくださる皆様の言葉が
心に染みすぎて
寝る前に読むと涙が出そうになってましたずっと（笑）
 
 
私は今の場所で頑張れば
見ててくれる人がこんなにもいるんだなって
改めて思えました
 
本当にありがとう！
 
 
存在感出せるように頑張ります！
 
ゆっくり、自分のペースで
少しずつ色んな目標を叶えていけたらいいな
 
一緒に登っていきたいなあ
 
 
 
 
 
 
 
 
質問返し◎
 
Q.やってみたい髪型とかある？
 
A.たくさんあります！せっかく髪もめちゃくちゃ長いし、たくさんヘアアレンジしたいな〜って思っておるよ☺︎握手会でやろうと思うから似合わなくても笑わないでね(.. )（笑）
 
Q.コスメの選び方は？
 
A.リップとかチークの色物は割と自分の好みで決めますが、他のアイテムとかは、メイク動画を参考にすることが多いです。それでも人に合う合わないがあるので、実際にお店で付けてみて〜って感じです(¨̮ )まだまだ欲しいものが沢山あります( .. )
 
Q.握手会タメ口でいい？
 
A.全然！むしろタメ口で来てください！！私も多分すごくテンションがあがるから、タメ口になってしまうかもですがどうかお許しを(.. )楽しみ〜
 
 
 
 
 
 
 
 
皆様に報告です
 
 
初めてのソログラビア、
アップトゥボーイさんにて
今日撮影していただいてきました！
 
 
目標にしていたことが一つ叶ったよ（ ｉ _ ｉ ）
 
決まった時にすぐに皆様に言いたいって
思ってたから、
撮影した今日、ブログで本当に幸せ
 
 
絶対に見て欲しい！！
私ならではのグラビアになってます！
 
オフショットは発売されてから載せるね☺︎
 
 
もっと頑張る！ここから！
 
 
 
 
 
 
 
１枚目 かわいい妹達に囲まれてるの図。幸せ
 
２枚目 だいすきはづ〜とても優しい子なんです本当に
 
３枚目 ももっぴ〜テンションがおかしくなるとこめちゃくちゃすき
 
 
載せたい写真たくさんあるんだけど容量が、、ごめんね
 
 
 
 
 
 
 
25日に開催されていた、
東京ガールズコレクション、見学させて頂いていました
 
乃木坂に入る前に
TGC、ガールズアワードと
２度ファッションショーは見たことがありました
 
加入する前と見た気持ちは全然違うものがあった
 
自信がなくてあまり口には出せないけど
私の憧れはここにも１つあるんだなって
改めて思えました。
 
先輩方はアイドルも、モデルさんも、
両方の姿を見ていると全然違って
ここではもうモデルさんなんだなって
 
憧れの気持ちが強くなりました。
 
 
いつか夢が叶うように
日頃から頑張ります！！！
 
 
 
 
 
 
 
昨日、ももっぴの首に充電器が巻きついてたから、
充電器！って言ったら
巻き付きガール( ¨̮ )
とか言って１人で爆笑してた
 
テンションおかしすぎ( ¨̮ )（笑）
 
 
 
 
 
 
 
握手会、追加分含め18部全ての部、
完売したと聞きました。
 
本当にありがとうございます（ ｉ _ ｉ ）
 
 
嬉しい！頑張ります！
私もすごい楽しみで
めちゃくちゃ楽しむから
皆様にも楽しんでいただきたいなあ
 
あ、
握手会でもツンデレしてくれるの〜？
ってコメント結構ありました( ¨̮ )（笑）
 
やるかな〜やらないかな〜
来てのお楽しみ（笑）
 
皆様に会ったらデレになる気しかしない( ¨̮ )（笑）
 
 
まずは４/１の全国握手会！初大阪！
 
私のことを少しでも気になってくださってる方がいるとしたら
会いに来てみて欲しいな
 
ペアがすごく気になる毎日です、、わくわく◎
 
 
 
 
 
 
 
最後に告知させて下さい！
 
３/２１ OVERTUREさん
 
３/２４ Samurai ELOさん
 
３/２４ B.L.T.さん
 
 
発売中です！
 
 
４/６ Top Yellさん
 
４/８ UTB+さん
 
４/１０ MARQEEさん
 
４/２２ BRODYさん
 
４/２２ アップトゥボーイさん
 
 
よろしくお願い致します！
絶対に見て欲しい！！
 
 
 
 
 
 
雰囲気違う２枚〜
下は光がめちゃくちゃ当たって失敗の図
 
 
 
ばいばい
 
またねっ
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
今は与えていただいてる役割を全力でこなしていきます、待っててね
 
やりたいことリストがひとつ叶ったよ
 
握手会、やっとありがとういえる
たのしもうね( ¨̮ )
 

れんたんれんたんれんたんたん、駄菓子屋れんたんおいしいよ！ 岩本蓮加

パステルの季節 吉田綾乃クリスティー
