2017.02.07 22:42

みなさまこんにちは！

乃木坂46３期生の

梅澤美波です！

いちごひとぱっく食べても

全然足りない

いちご不足( .. )

季節が終わっちゃう前に

もっと食べなきゃ！！！

牛タン食べに行きたいなあ〜

もうすぐ高校入試かな？？

勉強とっても辛いと思います

でも、頑張れば結果がついてくるはず！！

ラストスパート！

一緒に頑張りましょう（ ｉ _ ｉ ）！！

プリンシパルのリハーサルのとき

ももっぴ

質問〜コーナー☺︎

Q.前回のだ〜れだの写真の答えは？

A.間違えてる方たくさんいました！！

左から、りりあ、みなみ、あやのです！！

正解した方おめでとうございます☺︎

Q.緊張しますか？

A.めちゃくちゃ緊張しいです。

足の震えが止まらなくなります。

心臓の鼓動もたぶん人よりおっきいです（笑）

どっくんどっくんする

Q.甘党？辛党？

A.選べない〜( .. )

甘いもの大好きだし、辛いのも大好きです、、。

両方ほしい欲張り人間( ¨̮ )

Q.使っているメイク道具は？

A.オススメは、RMKさんの下地、ファンデーションです！！

薄づきだしキレイに見えます☺︎

プチプラだと、Candy Dollさんのホワイトピュアパウダー、めちゃくちゃ気に入って使ってます！！崩れにくいしキレイに見えます☺︎

コスメで１番好きなリップは、

YSLさんのヴォリュプテシャインとか匂いも色もめちゃくちゃ可愛いし、

ジルスチュアートさんのオイルルージュは見た目も可愛いし好きです( ¨̮ )

Q.好きなものは最初に食べる？最後？

A.絶対最後です〜

楽しみはとっておきたい( ¨̮ )

今回は終わります◎

いちごのけーき！！

今日は白石麻衣さんの

写真集発売日☺︎

予約していたのが届いて

公演始まる前に見ました( ¨̮ )

本当に素敵すぎました。

頑張れます本当に。

みなさん見ましたか〜？？☺︎

プリンシパル真っ只中です

すぐ下書き書き始めるのですが、

今回は今日書きました

プリンシパルについて触れるか迷いました

けど、やっぱり書きます

千秋楽終えたらまたちゃんと書くけど

今日は伝えたいことを

見に来てくれた日に

2幕に出られなくてごめんなさい

それが本当に悔しかった

お忙しい時間を割いて

来てくださってるのに

期待に応えられなくて本当に悔しかった

３幕の幕が開くと

青と水色のサイリウム振って下さる方が

たくさんいて

たくさんたくさんいて

笑顔でこっちを見てくださってて

サイリウム振ったり

タオルあげたり

うちわやボードに名前書いてくださってたり

サイリウム、うちわ、タオル、ボード

全部見えてます

後ろの後ろまで見えてます

気づくから。

ずっと見ててね

本当に支えです！

ありがとうの言葉じゃ足りないくらい

絶対にまた２幕に立ちます

もっともっと頑張るから

諦めたくない

諦めない！

壁なんてぶち壊す！

舞台への気持ちはめちゃくちゃ強い！

一緒に１歩ずつ進んでください

こんな私にも出来るんだよって

見ててください！

折り返し地点。

残り半分。

後悔しないように突っ走りたい！

上手く伝えられないけど、

めちゃくちゃ気持ちは強いし熱いです！

目指せ三役制覇！

こんな大事言って

こいつ大丈夫かなんて思わないでください

叶えたい！

みなさんを幸せにしてみせます！

寒い中、お忙しい中、

時間を割いて来て下さり

本当にありがとうございます。

はやく握手会したいなあ

みんな本当に頑張ってます

ほっぺすごい伸びるの

全然写真なくてごめんね

じゃあまたねっ

出られないことが当たり前にしたくない

このままプリンシパルは終われません

見ててください
